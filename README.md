# Telegram-style bottom navigation

Пример статической страницы с нижним меню в духе Telegram. Включает фиксированный таббар, иконки с фирменным голубым акцентом и подсветку активного состояния.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер, например:

```bash
python -m http.server 8000
```

После этого откройте [http://localhost:8000](http://localhost:8000).
